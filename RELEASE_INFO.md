# Echo Music v5.2.2 Changelog

- The Service Uptime screen now checks all YouLyPlus fallback servers if the primary one is offline to ensure accurate status reporting.
- Hid the miniplayer from the Service Uptime screen.
- Fixed an issue where an unstyled background strip appeared at the bottom of the Commits and Changelog screens.
- Improved the Service Uptime screen layout and added a custom offline message when JioSaavn reaches its daily limits.
- Fixed an issue where the loading animation pill was slightly off-center before the song's total duration was fetched.
- Added "Loading" text next to the buffering animation in the codec indicator pill.
- Added a loading animation in the codec indicator pill when the player is buffering.
- Hid "Hide player thumbnail", "Thumbnail corner radius", and "Crop album art" options in Appearance settings when Player Background Style is set to Apple Music.
- Fixed invisible play/pause button in mini player on dynamic backgrounds in light mode.
- Removed empty bottom strip in the Commits screen.
